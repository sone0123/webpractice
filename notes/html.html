HTML

HTML (HyperText Markup Language) とはコンテンツの構造を定義するマークアップ言語である。

目次

  • HTMLの構造
  • 主なHTML要素
  • 参考文献

HTMLの構造

HTMLファイル（.html）は次のような構造を持つ：


<!doctype html>
<html lang="ja">
    <head>
        <meta charset="utf-8" />
        <meta name="viewport" content="width=device-width" />
        <title>テストページ</title>
    </head>
    <body>
        <h1>テストページ</h1>
        <p>これはテストページです。</p>
        <img src="" alt="テスト画像" />
    </body>
</html>

文章や見出し，画像などのコンテンツは開始タグと終了タグで囲まれるか，自己終了タグで表されHTML要素と呼ばれる。


<要素名 属性1="値1" 属性2="値2" ...>
    コンテンツ
</要素名>

<要素名 属性1="値1" 属性2="値2" ... />

利用可能なすべてのHTML要素は HTML 要素リファレンス - HTML | MDNを参照のこと。

利用可能なすべてのHTML属性はグローバル属性 - HTML | MDNおよび HTML 属性リファレンス - HTML | MDNを参照のこと。

主なHTML要素

ここではよく使われるHTML要素について簡単に説明する。アクセシビリティの観点から各コンテンツに対して適切なHTML要素を使用するべきである。

<!DOCTYPE>
文書型宣言である。「ほとんど役に立たないが，それでも必要な」要素である。
<html>
ページのすべてのコンテンツを囲む要素である。言語を設定するlang属性を持つ。
<head>
ページの閲覧者に表示しないコンテンツを含む要素である。
<meta>
ページのメタデータを定義する要素である。
charset
ページの文字エンコーディングを定義する属性である。
nameとcontent
ページのメタデータを定義する属性である。
name="viewport"
ページのビューポートを定義する値である。
name="author"
ページの作成者を定義する値である。
name="description"
ページの説明を定義する値である。
propertyとcontent
Open Graph Protocolのプロパティを定義する属性である。
property="og:title"
ページのタイトルを定義する値である。
property="og:description"
ページの説明を定義する値である。
property="og:image"
ページの画像を定義する値である。
<link rel="" href="">
ページと外部リソースとの関係を定義する要素である。
rel="icon"
ページのアイコンを定義する値である。
rel="apple-touch-icon"
ページのApple Touch Iconを定義する値である。
sizes="167x167"
iPad Proの高解像度レティナ画面用のアイコンを定義する値である。
sizes="180x180"
3倍の解像度のiPhoneのアイコンを定義する値である。
sizes="152x152"
レティナでない端末のアイコンを定義する値である。
sizes=""
2倍の解像度のiPhoneおよびその他の端末のアイコンを定義する値である。
rel="stylesheet"
ページとスタイルを定義する値である。
<script src="" defer>
ページにJavaScriptコードを埋め込む要素である。
<title>
ページのタイトルを定義する要素である。
<body>
ページの閲覧者に表示されるコンテンツを含む要素である。
<header>
ページのヘッダーを定義する要素である。
<nav>
ページのナビゲーション機能を定義する要素である。
<main>
ページの主要なコンテンツを定義する要素である。
<section>
ページのセクションを定義する要素である。
<article>
ページの記事部分を定義する要素である。
<aside>
ページの補足的な内容を定義する要素である。
<footer>
ページのフッターを定義する要素である。
<div>
ブロックレベルの意味的でない部分を定義する要素である。
<span>
インラインレベルの意味的でない部分を定義する要素である。
<h1>から<h6>
見出しを定義する要素である。数字が小さいほど重要な見出しである。
<p>
段落を定義する要素である。
<em>
強調されたテキスト（イタリック）を定義する要素である。
<strong>
重要なテキスト（太字）を定義する要素である。
<a href="">
リンクを定義する要素である。
href="https://example.com"
外部リンクを定義する値である。
href="page.html"
同じディレクトリへの内部リンクを定義する値である。
href="subdirectory/page.html"
サブディレクトリへの内部リンクを定義する値である。
href="../page.html"
親ディレクトリへの内部リンクを定義する値である。
href="#section-id"
id="section-id"属性を持つ要素へのリンクを定義する値である。
href="mailto:example@example.com"
メールアドレスへのリンクを定義する値である。
target="_blank"
リンク先を新しいタブで開く値である。
<ul>と<li>
順不同リストを定義する要素である。
<ol>と<li>
順序付きリストを定義する要素である。
<dl>と<dt>，<dd>
説明リストを定義する要素である。
<blockquote cite="">
ブロック引用を定義する要素である。
<q cite="">
インライン引用を定義する要素である。
<abbr title="">
省略語を定義する要素である。
<address>
連絡先情報を定義する要素である。
<sup>と<sub>
上付き文字と下付き文字を定義する要素である。
<code>
コードを定義する要素である。
<pre>
コードブロックを定義する要素である。
<var>
変数名のマークアップ定義する要素である。
<kbd>
キーボード入力のマークアップを定義する要素である。
<samp>
コンピュータの出力のマークアップを定義する要素である。
<time datetime="">
日時や時刻を定義する要素である。
datetime="2016-01-20"
単純な標準的な日付を定義する値である。
datetime="2016-01-20T19:30+01:00"
タイムゾーン付きの日付と時刻を定義する値である。
datetime="2016-W04"
特定の週番号を定義する値である。
<img src="" alt="" width="" height="" />
画像を定義する要素である。
<table>
テーブルを定義する要素である。
<tr>
テーブルの行を定義する要素である。
<td>
テーブルのデータセルを定義する要素である。
<th>
テーブルの見出しセルを定義する要素である。
colspan=""
列の結合数を定義する値である。
rowspan=""
行の結合数を定義する値である。
scope="col"
列の見出しを定義する値である。
scope="row"
行の見出しを定義する値である。
<caption>
テーブルのキャプションを定義する要素である。
<thead>
テーブルのヘッダーを定義する要素である。
<tbody>
テーブルの本体を定義する要素である。
<tfoot>
テーブルのフッターを定義する要素である。
<figure>と<figcaption>
キャプション付きの画像や図表などを定義する要素である。
<br>
改行を定義する要素である。
<hr>
水平線を定義する要素である。
<button>
インタラクティブなボタンを定義する要素である。
<form>
フォームを定義する要素である。
