CSS

CSS (Cascading Style Sheets) とはHTML要素を選択的にスタイルするためのスタイルシート言語である。

CSSの構造

CSSファイル（.css）は次のような構造を持つ：


h1 {
    color: red;
    font-size: 2.5em;
}

p {
    color: aqua;
    padding: 5px;
    background: midnightblue;
}

HTML要素やその属性などをセレクターと呼び，スタイルを適用したいセレクターに対してプロパティとそのプロパティ値を宣言する。


セレクター {
    プロパティ1: プロパティ値1;
    プロパティ2: プロパティ値2;
    ...
}

利用可能なすべてのセレクターはCSS セレクター - CSS | MDNを参照のこと。

利用可能なすべてのプロパティはCSS プロパティ - CSS | MDNを参照のこと。

CSSの使い方

外部CSSファイルの使用

.htmlファイルの<head>要素内に次の<link>要素を追加する。


<link rel="stylesheet" href="styles.css" />

内部CSSの使用

.htmlファイルの<head>要素内に次の<style>要素を追加する。


<style>
    h1 {
        color: red;
        font-size: 2.5em;
    }
</style>

インラインCSSの使用

HTML要素のstyle属性にCSSプロパティを指定する。この方法は可能な限り避けるべきである。


<h1 style="color: red; font-size: 2.5em;">見出し</h1>

セレクターの指定方法

要素型セレクター

HTML要素の名前を使ってスタイルを適用する。


h1 {
    color: red;
    font-size: 2.5em;
}

クラスセレクター

HTML要素のclass属性の値を使ってスタイルを適用する。


<h1 class="important">見出し</h1>
        

.important {
    color: red;
    font-size: 2.5em;
}

IDセレクター

HTML要素のid属性の値を使ってスタイルを適用する。


<h1 id="main-heading">見出し</h1>
        

#main-heading {
    color: red;
    font-size: 2.5em;
}

属性セレクター

HTML要素の属性の値を使ってスタイルを適用する。


/* <a> 要素で title 属性を持つもの */
a[title] {
    color: purple;
}

/* <a> 要素で href が "https://example.org" と一致するもの */
a[href="https://example.org"] {
    color: green;
}

/* <a> 要素で href に "example" を含むもの */
a[href*="example"] {
    font-size: 2em;
}

/* <a> 要素で href が "https://example.org" で始まるもの */
a[href^="https://example.org"] {
    color: orange;
}

/* <a> 要素で href が ".org" で終わるもの、大文字小文字を区別しない */
a[href$=".org" i] {
    font-style: italic;
}

/* <a> 要素で class 属性に "logo" という語が含まれているもの */
a[class˜="logo"] {
    padding: 2px;
}

擬似クラスセレクター

HTML要素の状態を使ってスタイルを適用する。


a:hover {
    color: red;
}

擬似要素セレクター

HTML要素の特定の部分を使ってスタイルを適用する。


p::first-line {
    font-weight: bold;
}

全称セレクター

すべての要素にスタイルを適用する。


* {
    margin: 0;
    padding: 0;
}

複雑セレクター

複数のセレクターを組み合わせてスタイルを適用する。


/* <li> 要素の中の <em> 要素 */
li em {
    color: rebeccapurple;
}

/* <h1> 要素の直後の要素である <p> 要素 */
h1 + p {
    font-size: 200%;
}

/* <h1> 要素の後のすべての要素である <p> 要素 */
h1 ˜ p {
    color: teal;
}

セレクターリスト

複数のセレクターをコンマで区切って一度にスタイルを適用する。


h1, h2, h3 {
    color: blue;
}

アットルール

特定の条件下でスタイルを適用する。


@media (width >= 30em) {
    body {
    background-color: blue;
    }
}

フォントの設定

フォントは可用性を保証されないので複数のフォントを指定することが推奨される。


p {
    font-family: "Trebuchet MS", Verdana, sans-serif;
}

既定のフォント

実際に使用される正確なフォントはブラウザ次第である。

  • serif
  • sans-serif
  • monospace
  • cursive
  • fantasy

ウェブセーフフォント

すべてのシステムで利用できるフォント（ウェブセーフフォント）の種類は限られている。

  • Arial
  • Courier New
  • Georgia
  • Times New Roman
  • Trebuchet MS
  • Verdana

ウェブフォント

ウェブ上で利用できるフォントファイルを指定してアクセスしたときにダウンロードさせることができる。


@font-face {
    font-family: "zantrokeregular";
    src:
        url("zantroke-webfont.woff2") format("woff2"),
        url("zantroke-webfont.woff") format("woff");
    font-weight: normal;
    font-style: normal;
}

html {
    font-family: "zantrokeregular", "Bitstream Vera Serif", serif;
}

グリッドレイアウト

グリッドは列と行およびそれらの間の間隔を指定してレイアウトを制御する。


.container {
    display: grid;
}

12カラムグリッドシステム

例として次の.htmlファイルによく使われる12カラムグリッドシステムを適用する。


<div class="wrapper">
    <header class="main-head">ヘッダー</header>
    <nav class="main-nav">
        <ul>
            <li><a href="">ナビ 1</a></li>
            <li><a href="">ナビ 2</a></li>
            <li><a href="">ナビ 3</a></li>
        </ul>
    </nav>
    <article class="content">
        <h1>メイン記事領域</h1>
        <p>
            このレイアウトでは，幅が 500 ピクセル以下の画面では，領域をソース順に表示します。グリッドとグリッド上のアイテムの配置を再定義することで，2 列レイアウト，3 列レイアウトへと変化していきます。
        </p>
    </article>
    <aside class="side">サイドバー</aside>
    <div class="ad">広告</div>
    <footer class="main-footer">フッター</footer>
</div>

12列のグリッドを作成し，各列に1fr単位のスペースを設定し，各行の開始位置にcol-startという名前を付ける。


.wrapper {
    display: grid;
    grid-template-columns: repeat(12, [col-start] 1fr);
    gap: 20px;
}

モバイルファーストで最も狭い画面ではHTML要素の順序のままグリッド全体が表示されるようにする。


.wrapper > * {
    grid-column: col-start / span 12;
}

次のブレイクポイントでは2列のレイアウトにする。


@media (width >= 500px) {
    .side {
        grid-column: col-start / span 3;
        grid-row: 3;
    }
    .ad {
        grid-column: col-start / span 3;
        grid-row: 4;
    }
    .content,
    .main-footer {
        grid-column: col-start 4 / span 9;
    }
    nav ul {
        display: flex;
        justify-content: space-between;
    }
}

さらに次のブレイクポイントでは3列のレイアウトにする。


@media (width >= 700px) {
    .main-nav {
        grid-column: col-start / span 2;
        grid-row: 2 / 4;
    }
    .content {
        grid-column: col-start 3 / span 8;
        grid-row: 2 / 4;
    }
    .side {
        grid-column: col-start 11 / span 2;
        grid-row: 2;
    }
    .ad {
        grid-column: col-start 11 / span 2;
        grid-row: 3;
    }
    .main-footer {
        grid-column: col-start / span 12;
    }
    nav ul {
        flex-direction: column;
    }
}
