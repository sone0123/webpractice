JavaScript

JavaScriptとはウェブページに複雑な機能を実装することを可能にする軽量なインタプリタ型プログラミング言語である。

JavaScriptの一般的な用途は，DOM (Document Object Model) APIを介して動的にHTML/CSSを変更し，UIを更新することである。

JavaScriptはクライアントサーバーコードとしても，サーバーサイドコードとしても使用される。

JavaScriptの使い方

内部JavaScript

HTMLファイルの</body>タグの直前に直接記述する。


<script>
    // JavaScriptコードをここに書く
</script>

外部JavaScript

.jsファイルを作成しHTMLファイルの</head>タグの直前で読み込む。


// JavaScriptモジュールとして読み込む場合
<script type="module" src="script.js"></script>

// HTMLの構文解析後かつDOMContentLoadedイベントの前に実行する場合
<script defer src="script.js"></script>

JavaScriptコードの書き方

基本的なJavaScriptコードの書き方について説明する。

書き始める前に

コードを書き始める前に，まず解決したい問題を簡潔に実行可能な小さな単位に分割することが重要である。

変数と定数

必要な変数と定数を宣言する。可能な限り定数を使用し，必要に応じて変数を使用する。


// 変数の宣言
let x;
let y = 10; // 変数の初期化

// 定数の宣言
const PI = 3.14159; // 定数は必ず初期化する必要がある

変数と定数の命名規則

変数と定数の名前付けは以下のルールに従うべきである：

  • アルファベットと数字とアンダースコアのみを使用する
  • 先頭にアンダースコアを使用しない
  • 先頭に数字を使用しない
  • 小文字で始まるキャメルケースを使用する
  • JavaScriptの予約語を使用しない
  • 分かりやすい名前を使用する

変数の型

変数には以下の基本的な型がある：

  • 数値型（number）
  • 文字列型（string）
  • 論理型（boolean）
  • 配列型（array）
  • オブジェクト型（object）

関数

関数は再利用可能なコードブロックである。


// 関数の定義
function functionName(parameter1, parameter2, ...) {
    // 関数f(parameter1, parameter2, ...)の本体
    return value; // 関数が値を返す場合
}

// 関数の呼び出し
functionName(value1, value2, ...);

メソッド

メソッドはオブジェクトのプロパティとして定義された関数である。


// メソッドの定義
const myObject = {
    myMethod: function() {
        // メソッドの本体
    }
};

// メソッドの呼び出し
myObject.myMethod();

ブラウザ組み込み関数

一部の関数はブラウザのコード（大半はC++のような低レベルのシステム言語で書かれている）を呼び出して使用することができる。


// random()関数は0から1までの乱数を生成するブラウザ組み込み関数である
const myNumber = Math.random();

無名関数

無名関数は名前を持たない関数である。定義と同時に呼び出される。


// 無名関数の定義と呼び出し
function(parameter1, parameter2, ...) {
    // 関数f(parameter1, parameter2, ...)の本体
};

アロー関数

アロー関数はES6で導入された新しい関数の記法である。


// アロー関数の定義と呼び出し
(parameter1, parameter2, ...) => {
    // 関数f(parameter1, parameter2, ...)の本体
};

関数のスコープ

関数内で定義されている変数や関数は内部にスコープ（適用範囲）を持ち，外部からはアクセスできない。すべての関数の外にある最上位のスコープをグローバルスコープという。

イベント

イベントはユーザーの操作やブラウザの状態変化など，ブラウザ内で発生する出来事である。

  • 選択する
  • クリックする
  • カーソルを合わせる
  • 特定のキーを押す
  • ページを読み込む
  • ページを閉じる
  • フォームを送信する
  • エラーが発生する

イベントリスナー

イベントリスナーは特定のイベントが発生したときに実行される関数である。イベントを持つHTML要素はaddEventListener()メソッドを使用してイベントリスナーを追加することができる。


// 特定のセレクターに一致する最初のHTML要素を取得する
const elementName = document.querySelector("selector");

// イベントリスナーの定義
elementName.addEventListener("event", (parameter1, parameter2, ...) => {
    // 関数f(parameter1, parameter2, ...)の本体
});

基本演算

算術演算子

各演算は通常の数学と同様である。

  • +: 加算
  • -: 減算
  • *: 乗算
  • /: 除算
  • %: 剰余
  • **: 累乗

インクリメント演算子とデクリメント演算子

  • ++: インクリメント演算子。変数の値を1増加させる。
  • --: デクリメント演算子。変数の値を1減少させる。

代入演算子

  • =: 代入
  • +=: 右辺を左辺に加算して代入
  • -=: 右辺を左辺から減算して代入
  • *=: 右辺を左辺に乗算して代入
  • /=: 右辺を左辺に除算して代入

比較演算子

比較演算子は値を比較して真（true）または偽（false）を返す。

  • ==: 等しい
  • !=: 等しくない
  • ===: 厳密に等しい（型も含めて）
  • !==: 厳密に等しくない
  • <: より小さい
  • >: より大きい
  • <=: 以下
  • >=: 以上

文字列

変数の埋め込み

文字列の中に変数の値を埋め込むにはテンプレートリテラルを使用する。


const name = "Alice";
const greeting = `Hello, ${name}!`;

改行

文字列内で改行するにはテンプレートリテラルを使用するか\nを使用する。


// テンプレートリテラルを使用する場合
const multiLineString = `This is a string
that spans multiple lines.`;

// \nを使用する場合
const multiLineStringWithNewline = "This is a string\nthat spans multiple lines.";

引用符

文字列を囲むには次のようにシングルクォート（'）またはダブルクォート（"）を使用する。


const singleQuoteString = 'This is a string with single quotes.';
const doubleQuoteString = "This is a string with double quotes.";

便利なメソッド

.length
文字列の長さを返す
[index]
指定したインデックスの文字を返す
.includes(searchString)
指定した文字列が含まれているかどうかを判定する
.indexOf(searchString)
指定した文字列が最初に現れるインデックスを返す。見つからない場合は-1を返す
.slice(start, end)
指定した範囲の文字列を抽出する
.toUpperCase()
文字列を大文字に変換する
.toLowerCase()
文字列を小文字に変換する
.replace(searchValue, newValue)
指定した文字列を新しい文字列に置換する
.split(separator)
文字列を指定した区切り文字で分割して配列を返す

配列

配列は複数の値を格納するためのデータ構造である。


const myArray = [element1, element2, element3];

便利なメソッド

.length
配列の長さを返す
[index]
指定したインデックスの要素を返す
.push(element)
配列の末尾に要素を追加する
.pop()
配列の末尾の要素を削除して返す
.shift()
配列の先頭の要素を削除して返す
.splice(start, deleteCount)
配列の指定した位置から要素を削除する
.unshift(element)
配列の先頭に要素を追加する
.indexOf(element)
指定した要素が最初に現れるインデックスを返す。見つからない場合は-1を返す
.slice(start, end)
指定した範囲の要素を抽出する
.join(separator)
配列の要素を指定した区切り文字で連結して文字列を返す
.map(callback)
配列の各要素に対して指定した関数を呼び出して新しい配列を返す
.filter(callback)
配列の各要素に対して指定した関数を呼び出して条件を満たす要素の新しい配列を返す
.toString()
配列を文字列に変換する

オブジェクト型

オブジェクト型とは関連する変数や関数型集合である。


const objectName = {
    member1Name: member1Value,
    member2Name: member2Value,
    member3Name: member3Value,
};

ドット記法とブラケット記法

オブジェクトのプロパティにアクセスするには次の2種類の記法を使用する。

  • ドット記法: objectName.memberName
  • ブラケット記法: objectName["memberName"]

this

thisはオブジェクトのプロパティやメソッド内で使用される特別なキーワードである。


const myObject = {
    myProperty: "Hello",
    myMethod: function() {
        console.log(this.myProperty); // "Hello"と出力される
    }
};

コンストラクター

コンストラクターはオブジェクトを作成するための関数である。大文字で始まる名前を持ちnewを使用して呼び出す。


// コンストラクターの定義
function ConstructorName(parameter1, parameter2, ...) {
    this.member1Name = parameter1;
    this.member2Name = parameter2;
    // ...
}

// コンストラクターを使用してオブジェクトを作成する
const myObject = new ConstructorName(value1, value2, ...);

条件文

if文


if (条件1) {
    // 条件1が真の場合に実行されるコード
} else if (条件2 && 条件3) {
    // 条件2かつ条件3が真の場合に実行されるコード
} else if (条件4 || 条件5) {
    // 条件4または条件5が真の場合に実行されるコード
} else {
    // すべての条件が偽の場合に実行されるコード
}

switch文


switch (expression) {
    case value1:
        // expressionがvalue1と等しい場合に実行されるコード
    case value2:
        // expressionがvalue2と等しい場合に実行されるコード
        break; // 次のcaseを実行せずにswitch文を終了する
    default:
        // 既定で実行されるコード
}

三項演算子


条件
    ? value1 // 条件が真の場合に返される値
    : value2; // 条件が偽の場合に返される値

ループ

forループ


for (初期条件; 条件; 最後に実行される式) {
    // ループの本体
}

for...ofループ


const elements = [element1, element2, element3];

for (const element of elements) {
    // elementsの各要素に対して実行されるコード
    if (条件) {
        break; // ループを終了する
    }
    if (条件) {
        continue; // ループの次の反復に進む
    }
    // continueが実行されない場合に実行されるコード
}

whileループ


while (条件) {
    // 条件が真の間繰り返し実行されるコード
    // 最終的に条件が偽になるようにする必要がある
}

do...whileループ


do {
    // 最低限1回は実行されるコード
} while (条件);

DOM操作

要素の参照


// 特定のセレクターに一致する最初のHTML要素を取得する
const elementName = document.querySelector("selector");
// 特定のセレクターに一致するすべてのHTML要素を取得する
const elements = document.querySelectorAll("selector");

要素の内容の変更


// 要素のコンテンツを変更する
elementName.textContent = "New text content";
// 要素の属性を変更する
elementName.href = "https://example.com";

要素のスタイルの変更


// 要素のスタイルを変更する
elementName.style.color = "red";

要素の追加


// 新しい要素を定義する
const newElement = document.createElement("div");
// 新しい要素の内容を設定する
newElement.textContent = "This is a new element.";
// 既存の要素に子要素として追加する
elementName.appendChild(newElement);

要素の削除


// 既存の要素から子要素を削除する
elementName.remove();

エラー処理


try {
    // エラーが発生する可能性のあるコード
} catch (error) {
    // エラーが発生した場合に実行されるコード
    console.error(error); // エラーをコンソールに出力する
    // 必要に応じてエラーを処理する
} finally {
    // エラーの有無にかかわらず常に実行されるコード
}
